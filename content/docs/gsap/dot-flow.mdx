## Props

### DotFlow

| Prop      | Type            | Default |
| --------- | --------------- | ------- |
| isPlaying | boolean         | -       |
| items     | DotFlow.Items[] | -       |

### DotFlow.Items[]

| Prop        | Type       | Default |
| ----------- | ---------- | ------- |
| title       | string     | -       |
| frames      | number[][] | -       |
| duration    | number     | 150     |
| repeatCount | number     | 1       |
