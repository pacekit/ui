## Props

### LiquidGlass

| Prop         | Type         | Default |
| ------------ | ------------ | ------- |
| width        | number       | 120     |
| height       | number       | 120     |
| borderRadius | number       | 12      |
| tintOpacity  | number (0-1) | 0.1     |
| blur         | number       | 2       |
