## Customizing

### Custom Overlay Color

Use `overlayClassname` to replace the fill color with any Tailwind background class.

<DemoPreview demoPath="demo/gsap/fillable-button/custom-overlay.tsx" />

## Usage

```tsx
import { FillableButton } from "@/components/gsap/fillable-button"
```

```tsx
<FillableButton>Click me</FillableButton>
```

## Props

### FillableButton

| Prop               | Type                                                                                    | Default     |
| ------------------ | --------------------------------------------------------------------------------------- | ----------- |
| `variant`          | `"default" \| "outline" \| "destructive" \| "secondary" \| "ghost" \| "link"`         | `"default"` |
| `size`             | `"default" \| "xs" \| "sm" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"default"` |
| `overlayClassname` | `string`                                                                                | —           |
